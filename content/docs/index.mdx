# Tech Docs Portal

사내 프로젝트의 기술 문서를 연도별로 관리하는 포털입니다.

## 연도별 프로젝트

- **2025** — Project Alpha (클라우드 인프라), Project Beta (인증 시스템)
- **2026** — Project Gamma (데이터 파이프라인)

## 사용 방법

1. 왼쪽 사이드바에서 연도 및 프로젝트를 선택하세요
2. 각 프로젝트에는 개요, 아키텍처, 가이드 문서가 포함되어 있습니다
3. `Ctrl+K`로 전체 문서를 검색할 수 있습니다
